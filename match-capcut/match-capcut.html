Cách tìm file draft_content.json nhanh

❖ Windows OS
%LOCALAPPDATA%\CapCut\User Data\Projects\com.lveditor.draft\

💡 Bấm Windows + R, dán đường dẫn trên rồi gõ Enter để mở thư mục quản lý dự án CapCut.

 macOS
~/Library/Containers/com.lemon.lvonewp/Data/Library/Application Support/CapCut/User Data/Projects/com.lveditor.draft/

💡 Mở Finder, nhấn Cmd+Shift+G rồi dán đường dẫn này vào.

📌 Sắp xếp theo ngày sửa đổi gần nhất (Date Modified) trong thư mục trên để tìm thấy đúng thư mục chứa dự án bạn vừa chỉnh sửa trong CapCut.
